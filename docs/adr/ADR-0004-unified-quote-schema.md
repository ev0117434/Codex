# ADR-0004: Unified Quote schema

- Status: Accepted
- Date: 2026-03-09

## Context
Биржи и рынки имеют разные форматы сообщений, что усложняет вычисления downstream.

## Decision
Ввести единый `Quote`-контракт после этапа normalizer.

## Rationale
- Упрощает spread calculation.
- Делает код тестируемым и предсказуемым.
- Снижает связность между коллекторами и потребителями.

## Consequences
- Плюсы: меньше ветвлений в business logic, выше читаемость.
- Минусы: требуется поддержка адаптеров при изменениях API бирж.
