# Roadmap v1.2+

## После стабилизации v1.1

### v1.2
1. Добавить новые биржи через адаптерный слой.
2. Добавить простое time-series хранилище (например, parquet/batch).
3. Добавить HTTP API для чтения последнего snapshot/health.

### v1.3
1. Визуализация (dashboard).
2. Alerting rules (stale spikes, collector downtime).
3. Полуавтоматическое управление списками подписок.

### v1.4+
1. Multi-host/HA архитектура.
2. Очередь сообщений между этапами (если SHM станет узким местом).
3. Полноценный replay исторических данных.

## Принцип развития
Сохранять обратную совместимость контрактов модулей, чтобы расширения были эволюционными.
