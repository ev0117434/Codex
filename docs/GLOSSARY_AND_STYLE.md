# Glossary and Style Conventions

## RU/EN glossary
- **Discovery (`symbol_discovery`)** — модуль получения активных торговых пар и формирования подписок.
- **Collector (`collectors`)** — модуль подписки на биржевые WS-потоки и доставки raw events.
- **Normalizer (`normalizer`)** — модуль приведения raw events к единому `Quote`.
- **Snapshot (`snapshot`)** — периодический файл-артефакт с рассчитанным spread.
- **Runbook (`operations runbook`)** — инструкция эксплуатации и восстановления.
- **Health check** — проверка работоспособности процесса/модуля.
- **Stale** — устаревшие данные (старше порога свежести).

## Style conventions
1. Названия модулей и метрик фиксируем на английском (`collector`, `spread_reader`, `snapshot_latency_ms`).
2. Описательный текст допускается на русском, но ключевые термины должны быть единообразны.
3. Для времени в контрактах используем `*_ns` и явно указываем `Unix epoch ns`.
4. Для внешних артефактов используем явное `version: 1` или эквивалентный заголовок версии.
